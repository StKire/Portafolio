T.E.S.S

Sistema de control de Inventarios multiplataforma (JavaFX, Html).

Desarrollado por: Kire Gomez

Fecha de Creación: 2024

Descripción: Sistema de control de inventarios multiplataforma, desarrollado en JavaFX y Html, con el objetivo de llevar un control de inventarios de una empresa, permitiendo el registro de productos, categorías, proveedores y ventas. El sistema cuenta con un sistema de autenticación y autorización de usuarios, y un sistema de reportes que permite visualizar la información de los productos y ventas registradas.

Tecnologías utilizadas: Java, JavaFX, Html, CSS, JavaScript, MySQL, Spring Boot, Spring Security, JWT.
